# 快速启动指南

## 📋 前置要求

- Node.js 16+ (推荐 18 或 20)
- Python 后端已启动在 `http://localhost:8000`

## 🚀 启动步骤

### 1. 进入前端目录

```bash
cd C:\Users\30835\Desktop\JChatMind\JChatMind-Python\frontend
```

### 2. 安装依赖

```bash
npm install
```

第一次安装可能需要 2-5 分钟。

### 3. 启动开发服务器

```bash
npm run dev
```

服务器启动后，访问：**http://localhost:5173**

## 🎯 功能使用说明

### Agent 管理

1. 点击 **"➕ 创建 Agent"**
2. 填写表单：
   - **名称**：Agent 的名称
   - **描述**：可选，简短描述
   - **系统提示词**：定义 Agent 的行为（必填）
   - **模型类型**：选择 DeepSeek/智谱AI/OpenAI
   - **模型名称**：具体模型，如 `deepseek-chat`
   - **温度**：0-2 之间，控制回复随机性
   - **记忆窗口**：保留的历史消息数量
3. 点击 **"保存"**

### 对话交互

1. 导航到 **"对话交互"** 页面
2. 从下拉列表选择一个 Agent
3. 在输入框输入消息
4. 按 **Enter** 发送（Shift+Enter 换行）
5. 点击 **"➕ 新建"** 创建新会话
6. 点击左侧会话列表查看历史对话

### 知识库管理

1. 导航到 **"知识库"** 页面
2. 点击 **"➕ 创建知识库"**
3. 填写知识库信息
4. 上传文档：
   - 点击知识库卡片上的 **"上传文档"**
   - 选择文件（支持 txt, md, pdf, doc, docx）
   - 设置分块参数
   - 上传后会自动生成嵌入向量

## ⚠️ 常见问题

### Q: 页面显示"请求失败"
**A**: 检查 Python 后端是否运行在 `http://localhost:8000`

### Q: Agent 创建失败
**A**: 确保填写了所有必填字段，特别是**系统提示词**

### Q: 文档上传无反应
**A**: 检查文件格式是否支持，文件大小不要超过 10MB

### Q: 消息发送后无回复
**A**: 
1. 检查后端日志查看错误信息
2. 确认 Agent 的模型配置正确
3. 检查 API 密钥是否有效

## 🎨 UI 主题切换

在浏览器控制台执行：

```javascript
// 切换到暗色主题
document.documentElement.setAttribute('data-theme', 'dark')

// 切换到亮色主题
document.documentElement.setAttribute('data-theme', 'light')
```
